# Starbound mod: Automatic Ship Doors

This mod makes the base game's seven racial ship doors open and close automatically for any player, NPC, or creature that approaches them.

Attaching proximity sensors to doors to enable automatic opening and closing is [problematic](https://github.com/rl-starbound/rl_bettercrew/blob/master/user-guide.md#what-counts-as-a-locked-door) in Starbound. This mod enables the seamless movement of all entities around ships (or, at least those ships equipped with ship doors) without such troublesome wiring.

The ship doors retain their input nodes, and if something is wired to a door's input node, that wiring takes precedence for control of the door.

This mod alters only the base game's racial ship doors. It does not alter the racial ship hatches, nor does it alter any objects provided by mods.

## Uninstallation

This mod can be added or removed at will.
